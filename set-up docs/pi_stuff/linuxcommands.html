sudo
What
Allows you to run a command as another user (usually root).
Why
Required for admin tasks like installing software or editing system files.
Example
sudo reboot
apt update
What
Refreshes your system's list of available packages.
Why
Keeps your Pi aware of the latest versions and security patches.
Example
sudo apt update
apt upgrade
What
Installs newer versions of your installed packages.
Why
Keeps your Pi running the latest software.
Example
sudo apt upgrade
hostnamectl
What
View or change the name of your Pi.
Why
Useful for naming your Pi in the Tangnet mesh.
Example
sudo hostnamectl set-hostname brand-node-01
whoami
What
Displays your current username.
Why
Confirm you're not root (or that you are!).
Example
whoami
pwd
What
Prints your current working directory.
Why
Know where you are in the filesystem.
Example
pwd
ls
What
Lists files and directories in the current folder.
Why
It’s how you see what’s there. Like peeking into a toolbox.
Example
ls -l /home/brand
cd
What
Changes the current directory.
Why
You need to move around to get anything done.
Example
cd /home/brand/Documents
nano
What
Opens a basic text editor in the terminal.
Why
Edit config files and scripts directly.
Example
sudo nano /etc/hostname
shutdown
What
Shuts the system down safely.
Why
Prevent file corruption when powering off.
Example
sudo shutdown now
reboot
What
Restarts the system.
Why
Applies changes, resets hung systems.
Example
sudo reboot
top / htop
What
Monitors system processes and performance.
Why
See CPU/memory usage, kill rogue processes.
Example
top
htop
