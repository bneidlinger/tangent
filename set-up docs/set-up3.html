üíæ Step 2: Flash the OS (From Your Laptop)

  • If not using the gear menu, you can also manually configure Wi-Fi:
    • After flashing, open the boot partition and create a file named wpa_supplicant.conf
    • Paste the following (change SSID and password):
      • country=US
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev

  ssid="YourNetworkName"
  psk="YourSecurePassword"
  key_mgmt=WPA-PSK
}
    • Save the file and eject the card safely.

üñ•Ô∏è Step 2.5: (Optional) Direct Access Troubleshooting

  • Connect a monitor via micro-HDMI and a USB keyboard directly to the Pi.
  • Power on the Pi and log in with pi / raspberry or your set password.
  • Check the IP address with:
    • hostname -I
  • If no IP shows, Wi-Fi might not have connected. Verify:
    • cat /etc/wpa_supplicant/wpa_supplicant.conf
  • To edit Wi-Fi config:
    • sudo nano /etc/wpa_supplicant/wpa_supplicant.conf
  • Reboot once set: sudo reboot

üîß Step 3: First Boot and SSH

  • Once networked, try SSH from your laptop or desktop:

  • If using Windows and want to support .local hostnames:
    • Install Apple Bonjour: Bonjour for Windows
  • Update system packages after SSH login:
    • sudo apt update && sudo apt upgrade -y
  • Install basic tools:
    • sudo apt install git python3 python3-pip tmux htop -y
