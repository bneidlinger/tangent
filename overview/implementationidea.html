From 1 Pi to 100-Node AI Cluster

üì¶ Phase 1: Single Pi Setup

  • Install Raspberry Pi OS (Lite recommended)
  • Set up SSH, static IP, hostname (e.g., tangnet-node-01)
  • Install llama.cpp, whisper.cpp, and Python (3.10+)
  • Test a basic model (e.g., Mistral 7B Q2_K) and confirm token output

üîó Phase 2: 5-Node Mini Cluster

  • Connect all Pis via Gigabit switch with DHCP or static IPs
  • Set up passwordless SSH between all nodes
  • Assign roles: e.g., LLM inference, RAG, embeddings, voice transcriber
  • Create a control script to broadcast jobs or prompts across nodes
  • Build a basic HTML dashboard for monitoring/logging

üß† Phase 3: 10-Node AI Stack

  • Deploy nodes with distributed services (e.g., ChromaDB, FastAPI, node-specific Whisper)
  • Introduce job queue manager (e.g., Celery + Redis or NATS)
  • Implement persistent logging and stats (e.g., Prometheus or flat JSON logs)
  • Wire up UI input to routed service calls (voice or typed prompts)

üåê Phase 4: 25-Node Regional Cluster

  • Modularize the cluster by assigning "zones" (voice, text, retrieval, compute)
  • Introduce redundancy + failover: heartbeat and health checks
  • Map the roles visually on a UI dashboard (SVG or 3D space)
  • Start integrating memory (e.g., email or note archive)

üöÄ Phase 5: 100-Node TANGNET GRIDCORE

  • Deploy full AI mesh with microservices for Whisper, RAG, LLM, DB, API
  • Each Pi runs a container or service, optionally load-balanced
  • Cluster-wide message bus (e.g., MQTT or NATS) with topic-based routing
  • Storage replication across nodes with S3-compatible sync or rsync schedule
  • Voice command layer + long-term memory engine
  • Offline-first capability with total cloud independence

üß¨ Bonus: Personality + Agent Layer

  • Assign personas or task AIs to specific nodes (e.g., Historian, Scout, Archivist)
  • Allow AI-to-AI messaging across nodes to coordinate tasks
  • Log insights in natural language summaries
TANGNET is an open experiment in decentralized AI creativity and control. Run wild.
