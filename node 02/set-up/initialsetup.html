tangent-node-02 (16GB Raspberry Pi 5) — Initial Setup

1. Hardware Assembly

  1. Unbox your Raspberry Pi 5 (16GB) and accessories:
    • Pi 5 board (16GB model)
    • Case (with fan/heatsinks)
    • USB-C 27W power supply
    • 128GB+ microSD card
    • Micro-HDMI cable
  2. Install heatsinks on the main chips (CPU, RAM) using the included thermal pads.
  3. Attach the case fan to the board:
    • Locate the white 4-pin FAN header (top-right near USB ports).
    • Remove the factory cap, plug in the fan’s connector (fits one way).
  4. Place the Pi into its case, close it up, and ensure nothing pinches the fan wires.
  5. Insert the prepped microSD card into the Pi’s slot (underside).

2. Imaging the microSD Card

  1. Insert the microSD card into your main computer.
  2. Open Raspberry Pi Imager (latest version recommended).
  3. Select OS: Raspberry Pi OS (64-bit) (Bookworm or latest stable).
  4. Before writing, click the gear ⚙️ icon (advanced settings):
    • Set hostname: tangent-node-02
    • Set username and password
    • Configure Wi-Fi (SSID, password, country)
    • Enable SSH (use password or public key auth)
    • Set locale, timezone, keyboard layout
  5. Write the image to the SD card. Eject safely when done.

3. First Boot & Initial Startup

  1. Insert the imaged SD card into the Pi 5.
  2. Connect monitor (micro-HDMI), keyboard, and mouse.
  3. Plug in the power supply to boot the Pi.
  4. Wait for first boot:
    • Desktop appears. The Pi should connect to Wi-Fi automatically.
    • Log in with your set username/password.
    • If prompted, complete any remaining setup (updates, display config, etc.).
  5. Test internet connection (open browser, try https://www.raspberrypi.com).
  6. Update packages (open Terminal): 
    sudo apt update && sudo apt full-upgrade -y
  7. Confirm SSH & VNC access:
    • SSH: ssh username@tangent-node-02.local or use Pi’s IP
    • VNC: Use RealVNC Viewer to connect to tangent-node-02.local:5900 or the Pi’s IP
  8. Record the Pi’s assigned IP address: 
    hostname -I
