🔥 AXE75 Router First-Time Setup: Rickified Edition

🧱 Step 0: Unbox & Prep

  • Place router in a central, elevated area. Avoid microwaves and ghosts.
  • Plug power in. Connect WAN port to modem or ATT gateway.
  • Boot it up. LEDs should turn on—Power, 2.4G, 5G, 6G, Internet.

🌐 Step 1: Login to the Router

Visit http://tplinkwifi.net on any browser
Create login password when prompted

🚀 Step 2: Quick Setup

  • Choose connection type (probably Dynamic IP or PPPoE for ATT Air)
  • Clone MAC if your ISP binds IP to it
  • Set SSIDs & Passwords for all bands — name your Wi-Fi like a boss (e.g., TangNet_6E)
  • Enable Smart Connect only if you want 2.4/5G merged (6G stays solo)

🧠 Step 3: Secure the Damn Thing

  • Change router admin password
  • Enable WPA3 (disable WEP/WPA garbage)
  • Create a Guest Network and isolate it
  • Disable WPS unless you're into network roulette

🔧 Step 4: Customize Network

  • Assign static IPs to your RPi units & main rig in DHCP settings
  • Enable IP & MAC Binding for those devices
  • Enable QoS, give main rig + Pis priority
  • Optional: Create a wireless schedule if you sleep like a normie

📡 Step 5: Mesh & Device Handling

  • Enable OneMesh if expanding later — futureproof!
  • Check connected clients under Network Map
  • Rename them like you're tagging your army: "Pi-Master", "Rigzilla", etc.

🛠️ Step 6: Final Touches

  • Update firmware ASAP — get the bugs out
  • Back up your config
  • Set auto-reboot once a week, 3am when you're passed out
  • Optional: Set up USB share or Time Machine if you swing that way

🏁 You're Done

Your network is now Rick-proof. Unless you open 47 ports and disable encryption. Don’t do that. Seriously.
