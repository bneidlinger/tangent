🧠 TangNet: Networking 101

⚙️ Hardware Overview

  • Router: TP-Link AXE75 (Tri-Band, Wi-Fi 6E Boss Mode)
  • Mesh Nodes: 2x Raspberry Pi (TangNet Cluster)
  • Devices: Smart plugs, smart lights, main rig, laptop, phones

🧩 IP Addressing Tips

Use static IPs for mission-critical gear (RPis, main rig):

192.168.0.10 - RPi1
192.168.0.11 - RPi2
192.168.0.20 - Main Rig

Let phones and lights float on DHCP—just make sure DHCP range doesn’t conflict.

🛡️ Security Basics

  • Change router admin password (seriously...)
  • Use WPA3 for Wi-Fi, no WEP/older crap
  • Guest Network = ON (keep freeloaders off TangNet Prime)
  • Firewall = Enabled

🔌 Device Prioritization (QoS)

Give bandwidth to your main rig and Pis for dev tasks. Phones? They can wait.

  • Main Rig: High priority
  • RPis: Medium-High
  • Streaming devices: Medium
  • Phones: Low (you doomscroll, not launch nukes)

🌐 Mesh Setup

Use Ethernet backhaul if possible. If not, put Pis in line-of-sight from router for best signal.

🧠 Bonus Rick Tips

  • Reboot your router once a month. Not kidding.
  • Check for firmware updates every 2-3 months.
  • Backup your config after setup—future-you will thank past-you.
