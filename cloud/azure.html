1. Create Your Azure Free Account

  1. Go to azure.microsoft.com/free and click **Start Free**.
  2. Sign in or create Microsoft account, verify email and phone.
  3. Add payment method (credit/debit); you get **$200 credit (+12 months free services)** :contentReference[oaicite:2]{index=2}.

2. Install Azure CLI

# macOS/Linux
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

az login
# Opens browser for authentication

3. Quick Test: Blob Storage + Function

  • Create storage and upload web file:
    az storage account create --name tangnetstorage --resource-group tangnet-rg --location eastus --sku Standard_LRS
    az storage container create --account-name tangnetstorage --name web --public-access blob
    az storage blob upload --account-name tangnetstorage --container-name web --name index.html --file ./index.html
  • Create Azure Function 'helloTangnet':
    Python example:
    def main(req): return \"Hello from Azure Tangnet!\"

4. Optional: Linux VM (Free Tier)

  1. Use a **B1s** or **B2s** Linux VM (750 hours free/month).
  2. Enable SSH port, then login:
    ssh azureuser@YOUR_VM_IP
After 30 days or credit use, services switch to pay-as-you-go; 55+ services stay free within limits :contentReference[oaicite:3]{index=3}.
