1. Create Your AWS Account

  1. Go to aws.amazon.com/free and click **Create a Free AWS Account**.
  2. Enter your email/password, select the **Free Tier** plan, add a credit/debit card.
  3. Verify by phone and choose a basic support plan.

2. Create IAM Admin User

  1. In AWS Console, open **IAM** â†’ **Add user** — name: tangnet-admin.
  2. Select **Programmatic access** + **AWS Management Console** login.
  3. Attach policies: AmazonEC2FullAccess, AmazonS3FullAccess, AWSLambda_FullAccess, AmazonAPIGatewayAdministrator.
  4. Turn on MFA and download credentials.

3. Install & Configure AWS CLI

# macOS/Linux
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o awscliv2.zip
unzip awscliv2.zip && sudo ./aws/install

# Configure
aws configure
# Enter access key, secret key, region (e.g., us-east-1), output=json

4. Quick Test: S3 & Lambda

  • Create a public S3 bucket:
    aws s3 mb s3://tangnet-test-bucket
    aws s3 cp index.html s3://tangnet-test-bucket --acl public-read
  • Create a Lambda function named helloTangnet returning `"Hello from Tangnet!"`, then front it with an API Gateway endpoint.

5. Optional: EC2 ARM Instance (Free Tier)

  1. Launch a **t4g.micro** (Graviton) Linux instance.
  2. Open ports 22 (SSH) and 5000 (if running local API).
  3. SSH in:
    ssh -i key.pem ec2-user@YOUR_INSTANCE_IP
