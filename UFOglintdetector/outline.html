Tangnet “Watch‑the‑Skies” Project

Rapid‑response glint detector, plus tight‑beam radio pinger for uncorrelated objects at GEO.
Codename inside the Blizzard Project List.

0 · Executive Snapshot

  • Detect sub‑second mirror‑glints (specular reflections) using a CMOS cam + Pi 5 + GPU classifier.
  • Filter against live satellite TLEs; flag “rogue” events to the Tangnet dashboard.
  • Respond with a 5.7 GHz narrow‑beam beacon (CW/BPSK31) pointed at the glint coordinates.

2 · Software Pipeline

  1. Capture daemon (Pi) – libcamera @ 25 fps → frame diff → save burst on ≥10 bright pixels.
  2. Astrometric solve – solve-field crop; RA/Dec in <2 s.
  3. ZMQ push to GPU box.
  4. CNN filter (GlintNet) – dismiss hot pixels/meteors.
  5. TLE cross‑match (Skyfield) – drop known sats.
  6. Unknown → SQL + Grafana; trigger beacon.py.

4 · Safety & Legal Checklist

  • FCC Amateur Extra or experimental permit (Part 5) for ≥20 W @ 5.7 GHz.
  • Rotator interlock ⟶ RF off if <8° elevation.
  • Duty‑cycle ≤10 %, side‑lobe E‑field < FCC exposure limits.
  • Publish beacon parameters on QRZ / AMSAT for transparency.
