⚛ Geiger Counter → Raspberry Pi Wiring

Board: RadiatronDIY‑v1 (blue PCB). Use the 3.5 mm jack tip for the TTL pulse.
Pi shown is a 40‑pin header; any free GPIO can be used (here we choose GPIO 17).


JACK Tip (Pulse) Sleeve (GND) 5 V IN dupont female dupont female Raspberry Pi 40‑pin GPIO17 GND

Wire list

  • Tip pad → Pi GPIO17 (pin 11) — counts arrive as 5 V pulses (≈150 µs).
  • Sleeve pad → Pi GND (pin 9) — shared ground.
  • Power the Geiger board via its barrel jack: USB‑A (Pi) → 5 V barrel cable or use a separate 5 V adapter.

Now run the Python RPi.GPIO script from earlier. Each click triggers an interrupt on GPIO17, giving live CPM readouts.
